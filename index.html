کلاینت غیرهمزمان aiovir

یک فریمورک پیشرفته، کاملاً غیرهمگام (Asyncio) و ساختاریافته برای پایتون جهت توسعه ربات‌ها، ابزارهای اتومیشن و استخراج داده با دسترسی بومی به لایه‌های سرویس virasty.com.

aiovir یه کتابخونه‌ست که به‌توی اجازه می‌ده با زبان پایتون و به‌صورت کاملاً غیرهمزمان، با پلتفرم اجتماعی virasty.com ارتباط برقرار کنی. با این کتابخونه می‌تونی ربات‌های خودکار بسازی، داده‌ها رو استخراج کنی، پست بزاری، کامنت بذاری، پیام بدی، تو اتاق‌های صوتی شرکت کنی و کلی کار دیگه!

چرا غیرهمزمان؟ خب، وقتی می‌خوای همزمان چندتا کار انجام بدی — مثلاً پیام‌های جدید رو چک کنی، پست جدید بزاری و همزمان به پیام‌ها پاسخ بدی — کتابخونه‌های معمولی (همگام) کل برنامه رو متوقف می‌کنن تا یه کار تموم بشه. ولی aiovir با استفاده از asyncio، همه کارها رو همزمان و بدون معطلی انجام می‌ده. یعنی ربات‌ت سریع‌تر، کارآمدتر و حرفه‌ای‌تر کار می‌کنه.

نصب کتابخانه

برای نصب کافیه از دستور زیر استفاده کنی:

pip install aiovir

اگر می‌خوای آخرین نسخه‌ی در حال توسعه رو نصب کنی:

pip install git+https://github.com/karbaladevir/aiovir.git

شروع سریع

با این کد ساده می‌تونی وارد حساب بشی و اطلاعات خودت رو ببینی:

import asyncio
from aiovir import VirastyClient

async def main():
    async with VirastyClient(session_file='bot.vir') as client:
        await client.login('09123456789', 'pass')
        me = await client.get_me()
        print(f'سلام @{me.username}')

asyncio.run(main())

نشست شما به‌صورت خودکار در فایل bot.vir ذخیره می‌شه و دفعات بعد نیازی به لاگین مجدد نیست.
